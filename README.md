# Little Calendar
This project is a simple desktop calendar app, using HTML, CSS, JavaScript, and ElectronJS.

<img width="237" height="252" alt="Little-Calendar-Picture" src="https://github.com/user-attachments/assets/cdb221d5-3563-487f-9162-0cbbc7087815" />

## How to Run

### 1. Clone repo
```
git clone https://github.com/amaya-samirah/Little-Calendar.git
cd Little-Calendar
```
### 2. Install Node.js
```
npm install
```
### 3. Run app locally
```
npm run start
```
